# CodeVector Product Browser Backend

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas

## Features
- Browse 200,000 products
- Category filtering
- Cursor-based pagination
- Consistent results while data changes

## API Endpoints

GET /api/products

GET /api/products?category=Books

GET /api/products?limit=20

GET /api/products?createdAt=...&id=...
